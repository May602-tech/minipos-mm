# MiniPOS MM

Offline POS and inventory PWA for small shops.

## Features

- Offline PWA installable on mobile
- Categories and products
- Sales and receipts
- Sales history
- Sale cancellation and item returns
- Receipt numbering like INV-1001
- Shop info on receipt
- Customers
- Expenses and reports
- Backup and restore with JSON
- Myanmar / English language switch

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Push to GitHub and import the repo to Vercel. Build command: `npm run build`, output directory: `dist`.
